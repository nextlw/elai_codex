## Aviso sobre tecnologia experimental

O Codex CLI é um projeto experimental em desenvolvimento ativo. Ainda não é estável, podendo conter bugs, funcionalidades incompletas ou sofrer alterações que quebram compatibilidade. Estamos construindo-o de forma aberta com a comunidade e recebemos com satisfação:

- Relatórios de bugs
- Solicitações de funcionalidades
- Pull requests
- Boa energia

Ajude-nos a melhorar abrindo issues ou enviando PRs (veja a seção abaixo para saber como contribuir)!
